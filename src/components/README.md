# Components Directory

Shared UI components used across the application.

## Structure

- **ui/**: shadcn/ui components (button, input, card, etc.)
- **layout/**: Layout components (Header, Footer, Sidebar)
- **common/**: Common reusable components (LoadingSpinner, ErrorBoundary, etc.)

## Naming Convention

- Use PascalCase for all component files (e.g., `Header.tsx`, `LoadingSpinner.tsx`)
- shadcn/ui components use kebab-case (e.g., `button.tsx`, `input.tsx`)
